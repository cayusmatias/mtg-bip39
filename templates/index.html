About The Project

This project offers an alternative way to store your BIP39 mnemonic phrases. By leveraging the Scryfall API, we translate your mnemonic into a list of Magic: The Gathering cards, providing an innovative and memorable method to secure your bitcoin wallets.

Early Stage Project

Please be advised that this project is in its initial stages and has not yet been extensively tested by the community. It is not recommended to use this application for managing accounts with actual balances.
