Welcome to the BIP39 Mnemonic Alternative Storage Project

Convert BIP39 to MTG Convert MTG to BIP39

This project offers an alternative way to store your BIP39 mnemonic phrases. By leveraging the Scryfall API, we translate your mnemonic into a list of Magic: The Gathering cards, providing an innovative and memorable method to secure your cryptocurrency wallets.

We are excited to announce that this project will be made publicly available on GitHub. This open-source approach allows anyone interested to contribute, review, and modify the code for their purposes.

Early Stage Project

Please be advised that this project is in its initial stages and has not yet been extensively tested by the community. It is not recommended to use this application for managing accounts with actual balances. Once the repository is available on GitHub, users are encouraged to download and run it locally with their data to ensure safety and privacy.
